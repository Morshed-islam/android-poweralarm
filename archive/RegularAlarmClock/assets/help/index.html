1 Usage

For quick setup follow these steps:

  1. Open Power Alarm
  2. Tap "Alarm" tab at bottom right corner to open Alarm view.
  3. Tap "Add" button at top right corner to create new alarm.
  4. Input alarm name, select which time alarm should go off.
  5. If you want a cyclical repeating alarm, tap "+" button in "Repeat Rule", then tap "Everyday" to select other rules.
  6. Tap "Save" at top right corner.
  7. Tap "Schedule" at bottom left corner to see when alarms will go off.
  8. You can also check when the next alarm will go off in notification bar.

Following instructions will help you understand how Power Alarm works as well as advanced features you may want to know.

In Power Alarm there are two important concepts; Alarm and Schedule. Alarm is the rule you create to generate Schedule, while Schedule is the actual time an alarm will go off. For example, if you create one Alarm to go off at 8:00 everyday, you will in Schedule see entries at 8:00 today, tomorrow, the day after tomorrow, and so on.

When you open Power Alarm the Schedule view is presented, where you can see when all alarms will go off for the following two weeks. You can also disable the next alarm only, without affecting the others in the series. Check Schedule for a brief introduction.

If you want to modify alarm rules, tap Alarm tab. The Alarm view is presented, where you can create/modify/delete alarm rules, as well as disable/enable alarm rules. Check Alarm for a brief introduction.

After you modify alarm rules, the Schedule view will be updated accordingly. Always go to Schedule view to make sure that the time at which the alarm should go off is correct after you make modifications to an alarm rule.

After alarm is set, the next alarm time will be shown in notification bar, as well as remaining time. Tapping the notification bar message will open Power Alarm. You can disable notification bar message in Preferences.

When an alarm goes off, you will find Dismiss and Snooze button in Alarm view. The Snooze button is very large by default, however you can choose to use a regular size Snooze button in Preferences. The Dismiss and Snooze buttons will be disabled for 2 seconds when alarm goes off, as well as after the device is taken out of a pocket, to prevent misoperation.

If you tap "Dismiss" button, the alarm is dismissed and the next alarm time will be set (if one exists).

If you tap "Snooze", the alarm will snooze and go off again 10 minutes later. Snooze time is configurable in Preferences. After alarm snooze, you will find a message in notification bar. Tap the message to dismiss the snoozed alarm, a dialog will popup to confirm.

If you neither dismiss nor snooze the alarm, it will be snoozed automatically after sometime. After 2 times of automatic snooze, the alarm will be dismissed. A message will be left in notification bar. Auto snooze times is configurable in Preferences.

2 Schedule

When Power Alarm is launched, the Schedule view is presented. You can go back to Schedule view by tapping "Schedule" tab when you are in Alarm view.

In Schedule view you will find a list of all alarms set to go off the following two weeks. The alarm times are ordered by the time they will go off. In addition, the very first entry shows the next alarm time as well as remaining time.

To disable/enable a schedule, just tap the button on the right. The disabled schedule will turn gray and will not go off. This feature is beneficial if you for example have created an alarm rule to go off at 7:00 everyday. Then it's possible to disable the alarm only for tomorrow morning, the alarms for the day after tomorrow and following days will still go off as planned.

After you modify alarm rules, the Schedule view will be updated accordingly. Always check Schedule view to make sure that the alarm time is correct after you have modified an alarm rule.

3 Alarm

Open Alarm view by tapping "Alarm" tab when you are in Schedule view.

Alarm view is where you can create, view, modify or delete alarm rules. All alarm rules are listed in Alarm view. Tap an alarm rule to edit it, press and hold to open context menu, where you can view/delete/clone it.

To create a new alarm rule, tap "Add" button at the top right corner.

To create a new alarm rule similar to an existing one, press and hold an existing alarm rule and then tap "Clone" from context menu.

3.1 Create/Edit Alarm

Name
Tap the text box to input alarm name. The alarm name will be shown when the alarm goes off. It can also be spoken out loud by Power Alarm if you enable "Say Alarm Name" feature in Preference view.
Time
Scroll to change alarm time.
Repeat Rule
Initially the repeat rule is "One Time", which means the rule is empty, and the alarm will only go off one time.
Tap "+" button to add a repeat rule, the edit rule menu is opened. Tap the list to select what kind of rule you want to govern your alarm, and edit the corresponding options to your liking, then press "OK". Under "Repeat Rule" you can at any time get back to the edit rule menu by tapping the name of the rule. Available rule types are:
  • Selected Dates: selected arbitrary dates.
  • Everyday: will go off everyday.
  • Day of Week: select from Monday to Sunday, will go off every week.
  • Day of Week in Month: select from Monday to Sunday, will only go off on selected numerical week(s) (first, second, third etc) of every month.
  • Day of Last Week in Month: select from Monday to Sunday, will only go off in selected one week (last, second to last, etc) of every month.
  • Day of Month: selected days of every month.
  • Day Month Range: selected day and month range.
  • Day Month Year Range: selected day, month, and year range.
  • Alarm X Off Y Days: select a start date, an X and an Y. Alarm will go off on the start start date, continue going off for X days, then pause for Y days, then go off for X days again in a cycle.
Tap "trash" button to delete repeat rule.
If more then one repeat rule exists, beginning with the second rule there will be a combining rule, tap "Plus" to select from:
  • Within: go off only on dates existing both in previous rules and this rule. e.g. [day1, day2, day3] Within [day1, day3, day4] will result in [day1, day3].
  • Plus: go off on all dates existing in previous rules and this rule. e.g. [day1, day2] Plus [day2, day3] will result in [day1, day2, day3].
  • Subtract: go off on dates existing in previous rules but not in this rule. e.g. [day1, day2, day3] Subtract [day3, day4] will result in [day1, day2].
Hence it's possible to combine more repeat rules to achieve more complex results.
Press and hold on a repeat rule and drag to move its order in rule list.
Delete After Expired
Delete the alarm rule after the alarm has gone off if the alarm is not configured to go off again.
Only on Workday
Will only go off on a workday if checked. This option is visible only if the Repeat Rule is not "One Time".
Ring
Whether to play the ringtone when alarm goes off.
  • Default: according to Preferences.
  • Ring: always play ringtone.
  • No Ring: don't play ringtone.
Ringtone
The ringtone to play when alarm goes off.
Vibrate
Toggle vibrate when alarm goes.
  • Default: according to Preferences.
  • Vibrate: always vibrate.
  • No Vibrate: don't vibrate.

Tap "Back" button at top left corner to exit without save.

Tap "Save" button at top right corner to save your new alarm rule.

4 Workday

Tap "menu" key in Schedule or Alarm view, then tap "Workday" to open workday list view. Tap on workday to edit, press and hold on workday to open context menu, where you can view/delete workday.

In create/edit workday view, you can input name for workday, and change workday date repeat rules just like in create/edit alarm.

5 Preferences

Tap "menu" key in Schedule or Alarm view, then tap "Preferences" to open preferences view. Here you can change various global settings. Some of the settings are described here:

Ring
Whether to play ringtone or not when alarm goes off.
  • Device Setting: according to device ringtone setting.
  • Ring: always play ringtone.
  • No Ring: don't play ringtone.
Ringtone
The ringtone to play when alarm goes off.
Say Alarm Name
If enabled, will speak the name of the alarm and current time when alarm goes off. At the moment this feature only supports speaking in English and requires TTS Engine as well as English language pack installed on Android to run correctly. This feature works on most Android devices, but there are some where it may not. If after enabling Say Alarm Name an error dialog is visible when alarm goes off, disable it.
Default to disabled.
Vibrate
Whether to vibrate when alarm goes off.
  • Device Setting: according to device ringtone setting.
  • Vibrate: always vibrate.
  • No Vibrate: don't vibrate.
Snooze Time
Time to wait before going off again after an alarm has been snoozed.
Max Auto Snooze Times
Number of times alarm will auto snooze if not dismissed/snoozed. If set to "2 Times", alarm will go off for a total of three times.
Large Snooze Button
If checked, will show a very large snooze button when alarm goes off. Otherwise a regular size snooze button will be shown.
Default to enabled.
Disable Button in Pocket
Disable Dismiss/Snooze button when alarm goes off and the device is located in your pocket. This feature was introduced to prevent misoperation. If the device was put in pocket and taken out by hand, your finger may touch Dismiss/Snooze button accidentally. The button will become active 2 seconds after you have taken the device out from your pocket.
Default to enabled.
Next Alarm Notification
Show next alarm message in notification bar if checked. Otherwise only alarming, snoozed and timeout alarm will be shown in notification bar.
Default to enabled.
Popup Notification
If checked will show popup notification when alarm is set/changed, or Alarm and Schedule was modified. Otherwise only error level popup notification will be shown (e.g. fail to create an alarm).
Default to enabled.
Show Inactive Schedules
Show schedules inactivated by Workday if checked. If you create an alarm and enable the "Only on Workday" option, the alarm will only go off on a workday. If this feature is disabled, you will not see schedules on workdays generated from the alarm. If this feature is enabled, you will be able to see schedules on workdays generated from the alarm. Although the schedule will not go off, you will still be able to disable it. If you change your workday and the schedule becomes active, the disabled schedule will still not go off.
Default to disabled. This feature may be useful if you change workday frequently.
Stream Channel
Which channel used to play alarm sound. Default is Alarm. If the ringtone didn't play on your device, try changing to a different channel.
TTS Stream Channel
Which channel used to play TTS sound (Say Alarm Name feature). Default is Music.
